# NotchMind 用户界面设计规范

## 设计理念

采用复古未来主义美学，结合现代 macOS 人机界面指南，创造既独特又实用的界面。

## 视觉元素

### 1. 颜色方案
- 主色调: 深蓝灰 (#1e1e2e) - 背景色
- 强调色: 青蓝色 (#74c7ec) - 活跃元素
- 辅助色: 淡粉色 (#f5c2e7) - 提醒/警告
- 文字色: 亮灰白 (#cdd6f4) - 正文文字
- 次要色: 浅灰色 (#6c7086) - 次要信息

### 2. 字体
- 主字体: SF Mono 或 JetBrains Mono (等宽字体)
- 尺寸: 10-12px，确保在小空间内清晰可读
- 行高: 1.2-1.4，增强可读性

### 3. 图形元素
- 像素艺术风格图标
- 动态光效和辉光效果
- 几何线条和点阵图案
- 半透明层叠效果

## 界面组件

### 1. Notch 控制面板
- 尺寸: 动态适配不同型号 MacBook 的 notch 尺寸
- 形状: 圆角矩形，与系统 notch 一致
- 内容: AI 工具状态指示器、通知气泡、快捷按钮
- 交互: 点击展开更多选项，悬停显示详细信息

### 2. 权限请求弹窗
- 样式: 半透明覆盖层，毛玻璃效果
- 布局: 居中显示，包含请求来源、权限描述、操作按钮
- 动画: 滑入滑出，淡入淡出效果

### 3. 状态指示器
- 形状: 圆形或方形图标
- 颜色编码:
  - 绿色: 工具在线
  - 黄色: 工具忙碌
  - 红色: 工具离线
  - 蓝色: 权限请求
- 尺寸: 适配 notch 区域的小空间

## 动画与交互

### 1. 视觉反馈
- 鼠标悬停时轻微放大和辉光效果
- 点击时颜色变化和缩放效果
- 状态变化时平滑过渡动画

### 2. 内容切换
- 从概览到详情的流畅过渡
- 通知显示和隐藏的动画
- 快捷操作的即时反馈

## 可访问性

- 支持 macOS 系统的可访问性设置
- 确保足够的颜色对比度
- 提供键盘导航支持
- 支持大字体显示模式

## 响应式设计

- 适配不同尺寸的 MacBook notch (13", 14", 16")
- 根据屏幕分辨率调整像素密度
- 适配深色/浅色模式切换
