# NotchMind 项目文档索引

欢迎来到 NotchMind 项目的完整文档集合。本项目旨在复刻 Vibe Island 软件的所有功能，创建一个 macOS 应用程序，将 MacBook 的 notch（刘海）区域转换为 AI 编码代理的控制面板。

## 开发阶段

该项目目前处于规划和设计阶段，使用以下开发文档套件指导实现：

1. **项目启动与规划阶段** - 确定核心功能和架构方向
2. **开发实施阶段** - 逐步实现各个功能模块
3. **测试与优化阶段** - 确保功能正确性和性能达标
4. **发布准备阶段** - 安全审计和版本优化
5. **维护与更新阶段** - 持续改进和功能扩展

## 核心目标

- 复刻 Vibe Island 的所有核心功能
- 保持内存占用低于 50MB
- 实现复古未来主义用户界面
- 支持多个主流 AI 编码工具
- 确保 macOS 平台的最佳实践
