# NotchMind 功能说明文档 v0.1

## 文档概述

本文档描述 NotchMind 应用中 Notch 控制面板的所有可见交互功能，帮助用户快速上手并充分利用应用的各项能力。

**适用范围**：v0.1 版本
**目标用户**：使用 AI 编程助手的 macOS 开发者

---

## 1. Notch 控制面板概述

NotchMind 利用 MacBook Pro 的 notch（刘海）区域作为常驻控制面板，提供 AI 工具状态监控和快捷操作功能。控制面板具有两种显示状态：**塌陷状态**和**展开状态**。

---

## 2. 交互功能详解

### 2.1 塌陷状态（默认状态）

当用户未与 Notch 区域交互时，控制面板以紧凑的塌陷状态显示：

#### 状态指示器

- **形状**：圆形，直径 12 像素
- **位置**：居中显示于 notch 区域
- **颜色编码**：

| 状态 | 颜色 | 含义 |
|------|------|------|
| 运行中 | 绿色 (#A6E3A1) | AI 工具正在执行任务 |
| 等待中 | 黄色 (#F9E2AF) | AI 工具空闲，等待新任务 |
| 错误 | 红色 (#F38BA8) | 工具连接异常或执行失败 |
| 权限请求 | 蓝色 (#74C7EC) | 等待用户授权 |

#### 动态效果

- **脉冲动画**：运行状态下，圆形指示器会有规律地放大缩小，表示工具正在活跃工作
- **抖动动画**：错误状态下，指示器会轻微抖动，提醒用户注意异常

#### 交互触发

| 触发方式 | 响应行为 |
|----------|----------|
| 鼠标悬停 | 300 毫秒后自动展开为展开状态 |
| 鼠标离开 | 1.5 秒后自动收起为塌陷状态 |
| 点击 | 立即展开为展开状态 |

### 2.2 展开状态（功能面板）

展开状态显示完整的控制面板，宽度 280 像素，高度 200 像素。

#### 标题栏

- **应用名称**：左侧显示 "NotchMind" 标题
- **关闭按钮**：右侧显示 "X" 图标按钮，点击后收起面板

#### AI 代理状态列表

面板中央显示已检测到的 AI 工具列表，包括：

- **状态圆点**：8×8 像素，颜色与主指示器保持一致
- **工具名称**：显示检测到的 AI 工具名称（如 Claude Code、Cursor、Windsurf）
- **状态文字**：显示该工具的当前状态（Idle / Running / Waiting / Error / Disabled）

#### 快速操作按钮

面板底部提供两个快捷操作按钮：

| 按钮 | 功能 | 样式 |
|------|------|------|
| **Activate AI** | 激活选定的 AI 代理开始工作 | 强调样式（.borderedProminent） |
| **Settings** | 打开应用设置面板 | 普通样式（.bordered） |

### 2.3 工具提示（Tooltip）

当鼠标悬停在塌陷状态的 Notch 区域时，会显示一个浮动提示框：

- **内容**：显示当前系统状态，如 "Status: Running" 或 "Status: Waiting"
- **样式**：半透明黑色背景，圆角矩形
- **位置**：显示在 Notch 区域的下方

## 4. 视觉规范

### 4.1 颜色方案

根据 UI 设计规范，控制面板采用以下配色：

| 用途 | 颜色代码 | 说明 |
|------|----------|------|
| 面板背景 | #1E1E2E (深蓝灰) | 主背景色 |
| 强调色 | #74C7EC (青蓝色) | 活跃元素、按钮 |
| 提醒色 | #F5C2E7 (淡粉色) | 警告信息 |
| 文字色 | #CDD6F4 (亮灰白) | 正文文字 |
| 次要文字 | #6C7086 (浅灰色) | 状态标签 |

### 4.2 动画参数

| 动画类型 | 持续时间 | 缓动函数 |
|----------|----------|----------|
| 展开动画 | 300ms | easeInOut |
| 收起动画 | 200ms | easeInOut |
| 悬停延迟 | 300ms | - |
| 离开延迟 | 1500ms | - |

### 4.3 面板样式

- **背景**：圆角矩形 (cornerRadius: 12)
- **边框**：1 像素灰色描边 (透明度 20%)
- **阴影**：黑色 20% 透明度，半径 10，Y 偏移 5

## 5. 功能模块说明

根据架构设计文档，Notch 控制面板与以下后端模块协同工作：

| 模块 | 职责 | 与 UI 的交互 |
|------|------|--------------|
| Notch Renderer | UI 渲染与交互 | 渲染本控制面板 |
| Agent Manager | 工具状态管理 | 提供 detectedTools 数据 |
| Tool Bridge | 工具连接管理 | 维护与 AI 工具的通信 |
| Permission Layer | 权限管理 | 处理权限请求提示 |

## 7. 常见问题

**Q1: 为什么不显示任何 AI 工具？**

> 请确保已启动支持 IPC 通信的 AI 工具（如 Claude Code），NotchMind 会自动检测并显示已连接的工具。

**Q2: 状态指示器一直闪烁正常吗？**

> 正常运行状态下，绿色指示器的脉冲动画是预期行为，表示 AI 工具正在工作中。

**Q3: 面板无法展开怎么办？**

> 尝试点击 Notch 区域强制展开，或重启 NotchMind 应用重置状态。
